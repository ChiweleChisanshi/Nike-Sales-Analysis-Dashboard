# Nike Sales Analysis Dashboard

## Overview

This project is a Tableau dashboard that analyzes Nike U.S. sales performance across different sales channels, products, states, and time periods.

The goal of the project was to use data visualization to understand how Nike sales perform across online, outlet, and in-store sales methods. The dashboard focuses on key business metrics such as revenue, operating profit, operating margin, and units sold.

## Project Goal

The main goal was to create an interactive dashboard that helps answer questions such as:

* Which sales channels generate the most revenue?
* Which channels have the highest operating profit?
* How do sales and profit vary by state?
* Which products perform best across different channels?
* How does performance change over time?
* How do price levels differ by channel?

* Sales channel performance

## Key Insights

Some of the main insights from the dashboard include:

* In-store sales generated the highest overall revenue.
* Online sales had the highest number of units sold.
* Outlet sales also contributed strongly to revenue and unit sales.
* Sales performance varied across states and regions.
* Product performance differed depending on the sales channel.
* Operating profit and operating margin showed that high revenue does not always mean the strongest profitability.
